wilson 是一款开源的，分布式，前后端分离，现代化的游戏服务器网页管理面板，基于Docker实现。



## 特色

- 完全可视化的控制面板
- 多语言国际化支持
- ELK日志分析
- Prometheus系统监控
- 实例资源弹性控制
- 角色API权限管理
- 多用户，子用户管理
- 开放API支持
- 分布式服务器管理





## 构建

进入项目根目录，执行如下命令

```sh
make all
```
