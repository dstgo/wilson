wilson 是一款开源的，分布式，前后端分离，现代化的饥荒联机版服务器集群管理网页面板，基于Docker实现。



## 特色

- 完全可视化的控制面板
- 多语言国际化支持
- ELK日志分析
- Prometheus系统监控
- 实例资源弹性控制
- API权限管理
- 子用户管理
- 开放接口支持
- 分布式服务器管理
- 可视化模组管理支持
- 多层世界支持





## 构建

进入项目根目录，执行如下命令

```sh
make all
```
