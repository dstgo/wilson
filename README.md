# wilson
![Static Badge](https://img.shields.io/badge/go-%3E%3D1.20-blue)![GitHub](https://img.shields.io/github/license/dstgo/wilson?color=red)



Englist|[简体中文](#简体中文)

Wilson is an open source, distributed, front-end and back-end separation, and a modern web management panel for server cluster management of Dont Starved Together, based on Docker.



## Feature

- Fully visual control panel
- Multilingual internationalization support
- ELK log analysis
- Prometheus system monitoring
- Elastic control of instance resources
- API permission management
- Sub user management
- Open interface support
- Distributed server management
- Mod management support
- Multi-layer world support



## How to contribute

3. Submit changes
4. Push changes
5. Submit a Pull Request
